# Como rodar

Para rodar este código basta:

## Iniciar um ambiente virtual

### Windows
``` bash
python -m venv venv
```
em seguida:
```bash
venv\Scripts\activate
```

### Linux
```bash
python3 -m venv venv
```
em seguida:
```bash
source venv/bin/activate
```

## Instalar as dependências
```bash
pip install -r requirements.txt
```

## Rodar o código
Agora basta rodar o código main.py

### Windows
```bash
python main.py
```

### Linux
```bash
python3 main.py
```
