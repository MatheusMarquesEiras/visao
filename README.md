# Como rodar

Para rodar este código basta:

## Iniciar um a,biente virtual

### Windows
``` bash
Python -m venv venv
```

### Linux
```bash
python3 -m venv venv
```

## Instalar as dependências
```bash
pip install -r requirements.txt
```

## Rodar o código
Agora basta rodar o código main.py
